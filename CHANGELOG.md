# Changelog

### Added
- Changelog and semantic versions.
- Raw TCP actors.
- Move to `package.yaml` and `hpack`.
- Type-safe asynchronous messages.
- Supervisors for `MonadUnliftIO` actions.
- Test suite.
- PubSub actor.
- Support for bounded PubSub subscribers.
